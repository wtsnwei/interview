    合并排序¶

    1、算法设计¶

    合并排序是采用分治策略实现对 n 个元素进行排序的算法。它是一种平衡、简单的二分分治策略，过程大致分为：

    1. 分治：将待排序元素分成大小大致相同的两个子序列。
    2. 治理：对两个子序列进行合并排序。
    3. 合并：将排好序的有序子序列进行合并，得到最终的有序序列。

    2、伪代码详解¶

    （1）合并操作¶

    为了进行合并，引入一个辅助合并函数 merge(list, low, mid, high)，该函数将**排好序的两个子序列** list[low:mid] 和 list[mid+1:hight] 进行合并。

    合并方法：设置三个工作指针 i、j、k 和一个辅助数组 temp_list。其中，i 和 j 分别指向两个待排序子序列中当前待比较的元素，k 指向辅助数组 temp_list 中待放置元素的位置。比较 list[i] 和 list[j]，将较小的赋值给 temp_list[k]同时相应指针向后移动。如此反复，直到所有元素处理完毕。最后把辅助数组 temp_list 中排好序的元素复制到 list 数组中。

    合并程序如下：

    public static int[] merge(int[] list, int low, int mid, int high) {
    int tempList[] = new int[high-low+1]; // 申请一个辅助数组
    int i = low;
    int j = mid+1;
    int indexTempList = 0;
    while(i<=mid && j<=high) {
        if(list[i] <= list[j]){
            templist[indexTempList++] = list[i++];
        } else {
            templist[indexTempList++] = list[j++];
        }
    }

    // 对子序列list[low:mid]剩余的继续处理
    while(i<=mid) {
        templist[indexTempList++] = list[i++];
    }

    // 对子序列list[mid:high]剩余的继续处理
    while(i<=mid) {
        templist[indexTempList++] = list[j++];
    }

    // 将合并后的列表恢复到list
    for(i=low, indexTempList=0, i<=high; i++) {
        list[i] = templist[indexTempList++]
    }
}

    将序列分成两个子序列，然后对子序列进行递归排序，再把两个已排好序的子序列合并。
